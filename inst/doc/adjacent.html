Adjustacent category logit models using brglm2

Ioannis Kosmidis

05 February 2019

bracl

The brglm2 R package provides bracl which is a wrapper of brglmFit for fitting adjacent category models for orginal responses using either maximum likelihood or any of the various bias reduction methods described in brglmFit. There is a formal equivalence between adjacent category logit models for orginal responses and multinomial logistic regression models (see, e.g. the Multinomial logistic regression using brglm2 vignette and the brmultinom function). bracl utilised that equivalence and fits the corresponding Poisson log-linear model, by appropriately re-scaling the Poisson means to match the multinomial totals (a.k.a. the “Poisson trick”). The mathematical details and algorithm on using the Poisson trick for mean-bias reduction are given in Kosmidis and Firth (2011).

Opinion on stem cell research and religious fundamentalism

The stemcell data set ships with brglm2. Agresti (2015, Section 4.1) provides a detailsed description of the variables recorded in this data set (see also ?stemcell).

library("brglm2")
data("stemcell", package = "brglm2")

Maximum likelihood estimation

The following chunk of code fits an adjacent category logit model with proportional odds and reproduces Agresti (2010, Table 4.2). Note that the intercept parameters are different because Agresti (2010, Table 4.2) uses different contrasts for the intercept parameters.

stemcells_ml <- bracl(research ~ as.numeric(religion) + gender, weights = frequency, data = stemcell,
                      parallel = TRUE, type = "ML")
summary(stemcells_ml)

The value of the log likelihood ratio statistic here is

(lrt <- deviance(stemcells_ml) - deviance(stemcells_ml_full))
#> [1] 6.321373

and has an asymptotic chi-squared distribution with

(df1 <- df.residual(stemcells_ml) - df.residual(stemcells_ml_full))
#> [1] 4

The p-value from testing the hypothesis that stemcells_ml_full is an as good fit as stemcells_ml is

pchisq(lrt, df1, lower.tail = FALSE)
#> [1] 0.1763991

hence, the simpler model is found to be as adequate as the full model is.
